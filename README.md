# GYSpecs
所有工具仓库的源

## 使用
> 添加源 
```
source 'https://github.com/Zhugy/GYSpecs.git'
```

> 添加需要使用的仓库
```
pod  'GYWeakTimer'
```


### 工具类


| 仓库名称 | 仓库说明 |
| :-----: | ---- |
| [GYWeakTimer](https://github.com/Zhugongying/GYWeakTimer.git) | 防止定时器循环引用  |
| [GYBaseKit](https://github.com/Zhugongying/GYBaseKit.git) | 基础组件 |
